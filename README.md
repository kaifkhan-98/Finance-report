# Finance-report
1. Project Title / Headline
Financial Analysis Dashboard | Power BI

2. Short Description
The Financial Analysis Dashboard is an interactive Power BI project built to analyze banking transaction data and monitor overall financial performance. It provides insights into transaction amount, customer segments, transaction status, state-wise performance, and monthly trends through interactive visualizations and KPIs. The dashboard helps stakeholders make informed business decisions using real-time financial insights.

3. Tech Stack
Power BI Desktop
Power Query
DAX (Data Analysis Expressions)
Excel / CSV
Data Modeling

4. Data Source
The dashboard is built using a banking transaction dataset containing customer and transaction details.

Dataset includes:
Transaction ID
Customer Name
Transaction Date
Transaction Type
Transaction Status
Customer Segment
State
Gender
Total Amount
Transaction Fee
Tax Amount

5. Features & Highlights
   
Financial Overview
Displays Total Amount, Total Transactions, Average Transaction Value, Total Fee, and Total Tax.
Compare current performance with the previous year.

Transaction Analysis
Analyze transactions by status, customer segment, transaction type, and state.
Monitor monthly financial trends and tax collection.

Interactive Filters
Filter data by Year, Occupation, Merchant Category, and other dimensions.
Dynamic visuals update instantly based on selected filters.
